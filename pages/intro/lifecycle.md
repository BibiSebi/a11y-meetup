# When to adopt accessibility?

- 🧑‍💼 **Requirement Engineering** - User Stories 
- 🫂 **UX** - text spacing, content distribution
- 🖍️ **UI** -  color contrasts, font
- 🖥️ **Development** - semantic HTML, CSS, JS
- 🔬 **Testing** - automated and manual
